## RL 청산 모델 아키텍처 (ScalpExitEnv)

```
관측 (Observation Space):
  - entry_pnl_pct: 현재 미실현 수익률 (%)
  - hold_minutes: 보유 시간 (분)
  - momentum_1m: 최근 1분 모멘텀
  - momentum_5m: 최근 5분 모멘텀
  - vol_ratio: 거래량 급증 비율
  - rsi_14: RSI
  - bb_position: 볼린저밴드 내 위치
  - strategy_type: 진입 전략 (0=news, 1=spike, 2=whale)

행동 (Action Space, Discrete 3):
  0 = HOLD (계속 보유)
  1 = TAKE_PROFIT (즉시 익절)
  2 = STOP_LOSS (즉시 손절)

보상 (Reward):
  HOLD: -0.001 (시간 비용)
  TAKE_PROFIT: realized_pnl (양수면 보너스)
  STOP_LOSS: realized_pnl (음수지만 빠른 손절 보상)
  타임아웃: -현재손실 * 1.5 (벌칙)
```
