Bubblemaps Token Viewer

토큰 홀더 분포 시각화 — 지갑 간 연결, 대량 보유자, 클러스터 분석

예시:
bubblemaps.io | DexScreener | pump.fun
🫧

토큰 주소를 입력하세요

Bubblemaps는 상위 250개 홀더를 버블로 시각화합니다.
버블이 클수록 많이 보유, 선으로 연결되면 지갑 간 전송 이력이 있습니다.

  1. 1. 체인 선택 (Solana, Ethereum, BSC 등)
  2. 2. 토큰 컨트랙트 주소(CA) 입력
  3. 3. "맵 보기" 클릭
  4. 4. 버블을 클릭하면 지갑 주소, 보유량, 클러스터 정보 확인

CA를 모르면 DexScreener나 pump.fun에서 토큰 이름으로 검색하세요
